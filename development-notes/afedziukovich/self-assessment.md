## Описание моей работы

Я разработал следующие UI-компоненты и страницы:

- **AppHeader** — выпадающее меню пользователя с клик-аутсайд, динамическими ссылками, кнопкой выхода
- **Login page** — форма входа с валидацией
- **Register page** — форма регистрации с валидацией
- **Profile page** — смена пароля с подтверждением
- **Dashboard page** — отображение прогресса пользователя (уровень, опыт, решённые задачи)
- **Users table / Таблица лидеров** — пагинация, сортировка по логину и уровню
- **Настройка Vitest** — установка @nuxt/test-utils, конфигурация vitest.config.ts
- **Unit-тесты** — тесты для login, register, categories, task, API
- **Responsive стили** — медиа-запросы, стили
- **Деплой на Vercel** — настройка и публикация проекта

Инициализация проекта, бэкенд, API, AI, CodeMirror, бургер-меню — реализованы моим тиммейтом Артёмом.

## 2 личных Feature Component

1. **Выпадающее меню пользователя в AppHeader** — сложный компонент с клик-аутсайд, состоянием авторизации
2. **Users table (Таблица лидеров)** — пагинация, сортировка, загрузка данных с API
