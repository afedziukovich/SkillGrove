# Описание работы

Я провел анализ требований и разработал диаграммы прецедентов, активностей и последовательностей, чтобы понять, как будут взаимодействовать компоненты системы. Также я занимался разработкой бэкенда, включая создание моделей сущностей, репозиториев и интерфейсов для работы с базой данных. Кроме того, я интегрировал в проект систему авторизации и аутентификации, используя middleware и JWT-токены. Для улучшения производительности и безопасности я использовал кеширование и систему валидации данных. Для frontend-части я разработал страницу для решения задач, и интегрировал в него редактор кода CodeMirror. Также я добавил тостовые оповещения и улучшения UX. Помимо этого, я занимался настройкой CI/CD-процессов с помощью GitHub Actions. Кроме того, я интегрировал в проект базу данных Supabase.

## Основные задачи и вклад

### 1. Разработка структуры проекта

Разработал структуру проекта, включая папки и файлы, необходимые для работы приложения.

- Определил основные компоненты и их взаимодействие;
- Настроил конфигурацию проекта;
- Создал базовые страницы и компоненты.

### 2. Настройка среды разработки

Настроил среду разработки, включая установку необходимых пакетов и настройку файлов конфигурации.

- Установил необходимые пакеты (Nuxt, Vue, Pinia);
- Настроил файлы конфигурации (nuxt.config.js, package.json);
- Настроил linting и форматирование кода.

### 3. Разработка бэкенда

Разработал бэкенд приложения, включая создание моделей сущностей, репозиториев и интерфейсов для работы с базой данных.

- Создал модели сущностей (пользователи, задачи, ответы);
- Разработал репозитории для работы с базой данных;
- Создал интерфейсы для взаимодействия с бэкендом.

### 4. Интеграция с ИИ-сервисом

Интегрировал приложение с ИИ-сервисом, включая создание эндпоинтов для отправки запросов и получение ответов.

- Создал эндпоинт для отправки запроса к ИИ-сервису для оценки решения задачи;
- Настроил получение ответов от ИИ-сервиса;
- Интегрировал ИИ-сервис с бэкендом приложения.

### 5. Разработка фронтенда

Разработал фронтенд приложения, включая создание страниц и компонентов для взаимодействия с пользователем.

- Разработал компоненты для взаимодействия с пользователем (формы, кнопки, тосты);
- Настроил маршрутизацию и навигацию между страницами.

### 6. Настройка кеширования и оптимизация

Настроил кеширование и оптимизировал приложение для улучшения производительности.
